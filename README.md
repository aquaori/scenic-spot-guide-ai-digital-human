# 景区导览服务AI数字人-前端

## 简介

本项目是第十五届中国软件杯大赛的A组题目：《景区导览服务AI数字人》的前端代码，主要包含以下内容：

1. 前台
    - 多模态交互：支持语音输入和文本输入，数字人能以语音、表情和口型同步的方式进行回答；
    - 智能问答与讲解：能够准确回答关于景区历史、文化、景点特色等常见问题；
    - 个性化推荐：可根据游客兴趣(如“对历史感兴趣”或“喜欢自然风光”)推荐不同的游览路线和讲解重点。
2. 管理后台
    - 知识库管理：管理员可上传、更新和维护景区的讲解词、文史资料、常见问题及答案等知识文档，作为数字人的知识基础；
    - 数字人形象管理：可配置数字人的外观、服装、声音等，使其更贴合景区文化特色；
    - 游客感受度报告：系统能分析交互记录，生成游客关注点分析、情感趋势报告及服务建议反馈给管理方；
    - 数据大屏概览：展示当日/本周服务人次、热门问答、游客满意度趋势等核心运营数据。

## 项目结构

项目采用 monorepo 多应用结构：

- `apps/tourist-web`: 游客交互端
- `apps/admin`: 管理后台
- `packages/shared-api`: 接口层与 Axios 实例
- `packages/live2d-assets`: Live2D 模型资源
- `packages/live2d-runtime`: Live2D 渲染运行时环境
- `packages/shared-types`: 共享业务类型
- `packages/shared-utils`: 通用工具
- `packages/ui`: 公共 UI 基础能力

## 部署

安装依赖：

```bash
npm install
```

分别启动各子包：

```bash
npm run dev:tourist
npm run dev:admin
```
